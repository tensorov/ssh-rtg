## Key Design Decisions
- Stateless: all state on disk ({data-dir}/services/*.json), no in-memory cache
- Opaque IDs: {host}-{port} used for file naming but never parsed back
- Stdlib only: net/http, encoding/json, os, time, path/filepath
- Background cleanup every 30s: removes services with last_seen > TTL*3
- Heartbeat does NOT trigger Traefik config regeneration (deferred to W1-T3)
- Lazy register: heartbeat creates minimal service if host not found
- Validation: host required, port 1-65535, proto "http"|"tcp", TTL defaults to 30
