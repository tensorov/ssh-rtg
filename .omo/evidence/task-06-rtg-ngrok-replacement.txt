## Key Design Decisions
- Stdlib only: strings.Builder for YAML, no gopkg.in/yaml.v3
- Atomic write: .tmp → rename(2) prevents partial writes on power loss
- HTTP service with Domain → HTTP router + HTTPS/TLS router + service
- TCP service (no Domain) → TCP router + service
- HTTP service without Domain → silently skipped
- invalid Proto → silently skipped
- safeName() sanitizes host:port to DNS-safe label (dash on dots/colons)
- Wired in 3 places: register handler, delete handler, cleanup goroutine (len(removed) > 0)
- NOT wired in heartbeat handler (no config change)
